# convert-n-sum-money

THE PURPOSE OF THIS APPLICATION IS TO GET MULTIPLE RECEIPTS VALUES AND CONVERT THEM FROM BGN AND RON TO EURO, 
  ALSO THE APPLICATION SUMS ALL THE RECEIPTS VALUES, TO GIVE YOU A TOTAL, SO YOU CAN GENERATE A BILL WITH THE TOTAL COST(SUM) OF THE RECEIPTS.

ALSO, THE APPLICATION SERVES AS A GOOD PRACTICE FOR LEARNING JAVASCRIPT, CSS & HTML.
